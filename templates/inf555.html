INF555: Interaction Design and Usability Testing

Description

Understand and apply user interface theory and techniques to design, build and test responsive applications that run on mobile devices and/or desktops.

USC Catalogue 2019-2020

Assignments
Week Description
Assignment 1 Research two non-profit organizations who could use a responsive website to help them.
Assignment 2 Conduct research and create your Persona.
Assignment 3 Create a list of User Scenarios, Use Cases, User Stories, and Features.
Assignment 4 Create the first draft of your site or app map using Sketch.
Assignment 5 Create the final draft of your site or app map.
Assignment 6 Create a User Flow or Flows for your primary users.
Assignment 7 Create your Home Page or Main Screen using Sketch.
Assignment 8 Create all your secondary pages/screens in Sketch.
Assignment 9 Finalize all your responsive wireframes for your site and apply visual design.
Assignment 10 Getting started on your prototype.
Assignment 11 Write a script for your prototype. Starting making the homepage or main screen interactive
Assignment 12 Create your first usability test and test it on a friend.
Assignment 13 Planning and conducting the research and testing.
